# 🚖 Uber Data Analysis Dashboard

## 📊 Project Overview

This project presents an interactive Power BI dashboard built to analyze Uber ride data and uncover meaningful business insights. The dashboard helps track ride bookings, revenue trends, trip performance, and customer behavior through dynamic visualizations and KPIs.

## 🎯 Objectives

* Analyze ride booking trends and patterns
* Monitor revenue and business performance
* Identify popular pickup and drop-off locations
* Evaluate trip distance and duration metrics
* Support data-driven decision making

## 🛠️ Tools & Technologies

* Power BI
* Power Query
* DAX
* Data Modeling
* Data Visualization

## 📈 Key Insights

* Total bookings and revenue analysis
* Peak booking periods identification
* Location-wise trip performance
* Trip distance and duration trends
* Interactive KPI monitoring

## 📌 Features

* Interactive filters and slicers
* Dynamic visualizations
* Business KPI tracking
* User-friendly dashboard design
* Actionable insights for stakeholders

## 🚀 Dashboard Highlights

* Booking Performance Analysis
* Revenue Tracking
* Location Analytics
* Trip Trend Analysis
* Customer Behavior Insights
